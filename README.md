# IOAI 2026

¡Bienvenidos!

Este repositorio ha sido creado para acompañar a los estudiantes en su preparación para la **International Olympiad in Artificial Intelligence (IOAI)**.

Aquí encontrarán proyectos, ejercicios, notebooks y material de estudio relacionados con los temas fundamentales de inteligencia artificial.

La idea es que este espacio sirva como apoyo durante el proceso de formación, práctica y entrenamiento rumbo a la IOAI.

¡Mucho éxito en esta etapa de preparación!
